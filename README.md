## Get Start
### build

```
$ docker-compose build
```

### start

```
$ sh start.sh
```

### open url

```
# frontend
$ open http://localhost:3000 # for  osx

or

$ xdg-open http://localhost:3000 # for linux


# backend api
$ open http://localhost:8020 # for osx

or

$ xdg-open http://localhost:8020 # for linux
```
